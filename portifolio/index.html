Semana 8 - Introdução ao CSS e Seletores

Objetivos: Compreender o que é CSS, sua sintaxe, formas de aplicação, especificidade e comentários.

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem para estilizar páginas web, separando conteúdo (HTML) de apresentação.

Sintaxe Básica

Seletor { propriedade: valor; }

Este texto usa estilos básicos.

Formas de Aplicar CSS

  • Inline: <p style="color: red;">Texto</p>
  • Interno: Dentro de <style> no HTML.
  • Externo: Arquivo separado (como este portfólio).

Especificidade e Cascata

position: static (padrão), relative, absolute, fixed. Propriedades: top, right, bottom, left, z-index. display: block, inline, inline-block. Float (básico).

Semana 12 - Flexbox Básico

Objetivos: Introduzir Flexbox para layouts flexíveis.

Conceito de Flexbox

Flexbox organiza itens em containers flexíveis. display: flex. Propriedades: flex-direction (row), flex-wrap (wrap), justify-content (center), align-items (center), flex-grow (1), flex-shrink (0), flex-basis (auto).
